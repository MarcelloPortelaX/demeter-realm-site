# Demeter Realm

Site institucional e experiência interativa do ecossistema Demeter.

A página apresenta os produtos DemeterKit, Demeter Dashboard e Demeter GIS
por meio de uma árvore procedural animada, com navegação responsiva e
recursos de acessibilidade.

## Tecnologias

- HTML
- CSS
- JavaScript
- Canvas
- SVG

## Executar localmente

python -m http.server 8080

## Status

Protótipo institucional em desenvolvimento.
